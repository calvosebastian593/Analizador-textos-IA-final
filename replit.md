# AI Text Insight App

## Overview

Fullstack text analysis app (Spanish UI). Takes any text and returns a 2-sentence summary, BERT sentiment classification, and topic tags. Uses only Hugging Face (free API key).

## Stack

- **Monorepo**: pnpm workspaces
- **Backend**: Node.js 24, Express 5, TypeScript
- **Frontend**: React, Vite, TailwindCSS, shadcn/ui, Framer Motion
- **AI**: Hugging Face Inference SDK (`@huggingface/inference`)
- **Sentiment**: BERT (`nlptown/bert-base-multilingual-uncased-sentiment`) with continuous scoring
- **Validation**: Zod (autogenerated from OpenAPI spec via Orval)

## Key Commands

- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas
- `pnpm --filter @workspace/api-server run dev` — run backend
- `pnpm --filter @workspace/text-insight run dev` — run frontend

## Environment Variables

- `HUGGINGFACE_API_KEY` — Hugging Face token (free, type "Read")

The `.env` file goes at the project root. Loaded by `dotenv` in `artifacts/api-server/src/index.ts`.

## Project Structure

- `artifacts/api-server/src/lib/llm.ts` — LLM for summary + tags
- `artifacts/api-server/src/lib/bert.ts` — BERT for sentiment
- `artifacts/api-server/src/routes/analizar.ts` — Main route, orchestrates both
- `artifacts/text-insight/src/pages/Inicio.tsx` — Main UI page
- `lib/api-spec/openapi.yaml` — API contract

## API Endpoints

- `GET /api/healthz` — Health check
- `GET /api/models` — Available models
- `POST /api/analyze` — Analyze text: `{ text, provider: "huggingface", model }`
